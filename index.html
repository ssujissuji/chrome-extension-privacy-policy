Bookmark Extension 개인정보처리방침

최종 업데이트: 2026년 5월 20일

Bookmark Extension은 사용자의 Chrome 북마크를 더 편리하게 확인, 검색, 정렬, 수정할 수 있도록 돕는 Chrome 확장 프로그램입니다.

1. 수집 또는 접근하는 데이터

이 확장 프로그램은 기능 제공을 위해 다음 데이터에 접근할 수 있습니다.

  • Chrome에 저장된 북마크 및 북마크 폴더 정보
  • 사용자가 확장 프로그램 안에서 설정한 폴더 색상 및 테마 정보
  • 사용자가 선택한 북마크를 열기 위한 탭 관련 정보

2. 데이터 사용 목적

접근한 데이터는 다음 목적에만 사용됩니다.

  • 북마크와 폴더 목록 표시
  • 북마크 검색, 정렬, 추가, 수정, 삭제
  • 폴더 색상 및 사용자 설정 저장
  • 사용자가 선택한 북마크 페이지 열기

3. 데이터 저장

사용자의 폴더 색상 및 테마 설정은 Chrome에서 제공하는 chrome.storage API를 통해 저장될 수 있습니다.

4. 데이터 공유

Bookmark Extension은 사용자의 북마크 정보, 설정 정보, 브라우징 정보를 외부 서버로 전송하거나 제3자에게 판매 또는 공유하지 않습니다.

5. 광고 목적 사용

Bookmark Extension은 사용자 데이터를 개인화 광고, 리타게팅 광고, 관심 기반 광고 목적으로 사용하거나 전송하지 않습니다.

6. Limited Use Disclosure

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
